# ONPS Macro

The **BEST** ONPS macro there is.

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/verypros/ONPSM/refs/heads/main/loader.luau"))
```

## Requirements

- A stable internet connection
- A good Computer/Laptop

## Features

- Auto Startup
- Auto Shift 
- Auto Random Failures (RF)
- Webhook Alerts
